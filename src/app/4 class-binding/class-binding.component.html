4. Class Binding


Class Binding se trata de una forma de vincular dinámicamente las clases CSS de un elemento HTML.


Útil para cambiar el estilo a elementos de un archivo HTML.


Sintáxis de Class Binding



En el archivo TypeScript se define una propiedad, es decir, se crea una variable y se define el tipo de dato:
En el archivo HTML , para llamar a la clase se usa la sintaxis
[class]= "className"
, donde className es el nombre de la clase que se creó en el archivo TypeScript.

Ejemplo Práctico



Se programaron tres botones que cambian el color del texto. Estos botones hacen uso de la clase
text_color
para cambiar el color, asignando el valor contenido en el evento
(click)


(click)=
text_color
= 'text-danger'"


En este ejemplo, el evento
(click)
llama a la clase
text_color
, y se le entrega el valor
text-danger
, esto hace que el texto se pinte de color rojo.

Este proceso se repite para el resto de botones, cambiando el valor de cada uno de estos.

Demostración

Texto que va a cambiar de color

Ejemplo Práctico 2



Se define la propiedad para activar o desactivar dinamicamente un boton, siendo el nombre de la clase
button_disable
siendo del tipo
boolean.

La propiedad button_disable se vincula a la clase CSS disabled del boton.

Un segundo boton permite alternar el estado de la clase button_disable entre true y false.
