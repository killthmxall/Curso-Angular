7. Directiva ngIf; else


La directiva ngIf y else es utilizada para mostrar u ocultar un elemento, siendo necesaria una condición booleana antes de su declaración.


ngIf junto con else permite renderizar un bloque alternativo si la condición no se cumple.


ngIf al tener la condición "false", no oculta el elemento, si no que lo elimina directamente del DOM.


Sintáxis de la Directiva ngIf



Se declara usando un asterisco antes de la directiva:
"*ngIf"

else se declara seguido de la directiva ngIf:
"*ngIf; else"


Ejemplo



En la clase principal del componente se modifica el método
mostrar_en_consola()
añadiendo los siguientes parámetros:

openAlert:boolean = false;

this.openAlert = true;

openAlert
cambia de
false
a
true,

mostrando el elemento alert que contiene la directiva *ngIf=openAlert


Se añade un nuevo botón "x" con el cual ocultar la alerta que se mostró.

Dentro de este elemento button, se establece un evento
(click)
para cambiar el valor
true
de
openAlert
a
false.
