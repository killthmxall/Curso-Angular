1. Interpolación


En angular, la Interpolación permite mostrar datos que han sido guardados en la clase.
Utiliza la sintaxis de doble llave, dentro se escribe el nombre de la variable:
{{ interpolacion }}


Se define una variable del tipo number en la clase
InterpolacionComponent
de el archivo
interpolacion.component.ts
El valor de esta variable se mostrará en el HTML de la aplicación, haciendo uso de las dobles llaves:
{{ interpolacion }}
