8. Directiva ngFor


La directiva ngFor se trata de una forma de iterar sobre una lista de elementos, es un bucle que repite el contenido de un array.


Sintáxis de la Directiva ngFor



Se declara dentro de un elemento
li
usando un asterisco para comenzar con la directiva:
"*ngFor"

seguido de las sentencias let ... of
"*ngfor= "let i of items"


Ejemplo



Es necesario crear una interfaz para establecer la estructura o el tipo de elementos que se guardarán en el arreglo de elementos.


En el componente TypeScript (ts) se deberá importar la interfaz anteriormente creada, para después llenarla de elementos que sigan la estructura establecida.


Se establece un arreglo de elementos o array denominado "personas", que parte de la interfaz Persona[].


Se definen varias personas que cumplan con la estructura de la interfaz persona.


Para acceder a los elementos del array creado en el archivo .ts, se debe almacenar los elementos en un bucle.


Para esto, se creará un listado
li
dentro del cual se establecerá la directiva
*ngFor.


La directiva *ngFor debe ir acompañada de las sentencias
let
y
of.


Se utiliza la directiva *ngFor para iterar sobre un arreglo llamado
personas
y crear una lista ul donde cada elemento li muestra el índice y las propiedades
nombre
y
edad
de cada objeto dentro del array.
