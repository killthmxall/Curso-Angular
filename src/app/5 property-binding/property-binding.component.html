5. Property Binding


Property Binding se trata de una forma de enlazar las propiedades del component TypeScript con las propiedades de los elementos HTML.


Es una forma unidireccional de enviar datos desde la lógica del componente hacia la vista.


Sintáxis de Property Binding



En el archivo TypeScript se define una propiedad, es decir, se crea una variable y se define el tipo de dato.


En este caso, la variable se nombra:
imagen
y su tipo de dato es:
string.


Además, se le otorga el enlace para obtener una imagen de stock.




En el archivo HTML , para llamar a la clase se usa la sintaxis
[elementoPropiedad] = "componentePropiedad"
con la propiedad HTML entre corchetes "[]", donde [elementoPropiedad] es el nombre de la propiedad HTML de dicho elemento, puede ser
src,
disabled,
value,
etc.


Mientras, "componentePropiedad" es la expresión o variable creada en el componente TypeScript.




En el ejemplo, la propiedad que se encierra dentro de los corchetes es:
[src]
, y se llama a la variable
"imagen"
creada en la clase del componente.
